Windows Media Player Plugins

Google Chrome

Windows Media Plugin for Chrome can be downloaded from
https://support.google.com/chrome/answer/95697?hl=en

Mozilla Firefox

Instructions on Installing the plugin for Mozilla Firefox
https://support.mozilla.org/en-US/kb/play-windows-media-files-in-firefox#w_installing-the-plugin

Windows Media Plugin for Mozilla Firefox can be downloaded from
https://support.mozilla.org/en-US/kb/play-windows-media-files-in-firefox

Safari

Flip4Mac plugin is needed to play Windows Media Video on Mac. The plugin can be downloaded from the following sites:

http://www.telestream.net/flip4mac/

http://windows.microsoft.com/en-us/windows/windows-media-components-quicktime
